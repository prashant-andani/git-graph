### :eyes: What's included
- Contribution calendar at repo level
- Total Commits so far, Commit counts by author
- Commit history

### :wink: Steps to Install

`npm i git-scene -S`

or

`yarn add git-scene`

In package.json, add

`scripts: { "git-scene": "git-scene" }`

#### Run

`npm run git-scene`

:wave: Thats It !!!!
