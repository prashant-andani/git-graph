# git-scene :tada:

Git-scene is a CLI tool that provides a stats of any local git repository.

#### What's included
* Contribution calendar at project level
* Total Commits so far, Commit counts by author


#### Steps to Install

`npm i git-scene -S`

or

`yarn add git-scene`

In package.json, add
`scripts: { "git-scene": "git-scene" }`

#### Run
`npm run git-scene`

Thats It !!!!
